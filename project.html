Project Details

Aim of this project is to create a smart floating robot that removes waste from the surface of water without help of any human. It uses a Raspberry Pi 5 for control and is powered by rechargeable batteries charged through solar panels, so that it can function for extended periods of time without requiring frequent recharge.A camera, along with a Faster RCNN based object detection system, allows the robot to detect trash in real time. Two small DC motors with propellers help the boat to move. When debris is detected, a conveyor gently lifts the waste and deposits it into a container on the robot. The design is simple and cost-effective while remaining efficient, making it an ideal choice for cleaning lakes, ponds, and other stagnant water areas. This robot is designed to safeguard marine life and promote a cleaner,more sustainable ecosystem.
